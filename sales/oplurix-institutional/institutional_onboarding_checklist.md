# EcoR Institutional Onboarding Checklist

## Before Payment

- confirm team size
- confirm language
- confirm taxa or protocol
- confirm whether adaptation is needed
- confirm onboarding format

## After Payment

- send delivery email
- send license terms
- send onboarding date options
- prepare the team package
- prepare the adapted template set if promised

## During Onboarding

- explain folder structure
- explain what each template is for
- show how to start with the workflow
- restate the support boundary
- capture any real blockers the team reports

## After Onboarding

- send recap email
- log any follow-up tasks
- confirm whether the team accessed the files successfully
