## Folder Guide

- `institutional_quote_log.csv`

- `institutional_intake_form.md`
  - discovery questions to scope the team need
- `institutional_scope_template.md`

- `institutional_quote_template.md`

- `institutional_license_terms.md`

- `institutional_onboarding_checklist.md`

- `institutional_followup_templates.md`
  - message templates for reply, quote, and follow-up

## Institutional Workflow

1. Receive interest from the contact form or direct email.
2. Log the lead.
3. Send the intake form.
4. Write the scope note.
5. Send a manual quote.
6. Confirm payment manually.
7. Deliver the license pack and onboarding steps.
8. Run onboarding.
9. Record any adaptation requests and completion notes.

## Important Boundary

This is not a self-serve product yet.
It is a lightweight, founder-led team offer.
