t-test apparié
Comparer 2 mesures sur les mêmes unités — données normales
Code R
t.test(avant, apres, paired = TRUE) # Ou avec un dataframe : t.test(valeur ~ temps, data = df, paired = TRUE)
Hypothèses
  • Différences (avant − après) normalement distribuées
  • Même unités mesurées aux deux temps
  • Indépendance entre les paires
Vérifiez avec shapiro.test(avant - apres). Si p < 0.05 → utilisez le test de Wilcoxon signé.
t-test de Student
Comparer 2 groupes indépendants — données normales
Code R
t.test(shannon ~ habitat, data = df) # Vérifier les variances d'abord : var.test(shannon ~ habitat, data = df) # Si p < 0.05 : variances inégales t.test(shannon ~ habitat, data = df, var.equal = FALSE) # Welch (défaut)
Hypothèses
  • Normalité dans chaque groupe (shapiro.test)
  • Indépendance des observations
  • Welch corrige automatiquement les variances inégales
Taille d'effet
library(effectsize) cohens_d(shannon ~ habitat, data = df) # d < 0.2 = petit | 0.5 = moyen | 0.8 = grand
ANOVA à un facteur
Comparer 3+ groupes — données normales, variances égales
Code R
model <- aov(shannon ~ habitat, data = df) summary(model) # Vérification des résidus par(mfrow = c(2,2)) plot(model) # QQ-plot + résidus vs ajustés
Post-hoc obligatoire si p < 0.05 : quels groupes diffèrent ?
Tests post-hoc
# Tukey HSD (recommandé, contrôle le taux d erreur) TukeyHSD(model) # Ou avec emmeans (plus flexible) library(emmeans) emmeans(model, pairwise ~ habitat)
Si les résidus ne sont pas normaux ou si les variances sont inégales → Kruskal-Wallis.
Régression linéaire
Relation entre diversité et variable continue — données normales
Code R
model <- lm(shannon ~ altitude, data = df) summary(model) # Plusieurs prédicteurs : model2 <- lm(shannon ~ altitude + temperature + couverture, data = df) # Sélection de modèle AIC : library(MASS) model_sel <- stepAIC(model2, direction = "both")
Vérification des résidus
par(mfrow = c(2,2)); plot(model) # Chercher : résidus centrés sur 0 # QQ-plot sur la diagonale # Pas de patron dans résidus vs ajustés
Test de Wilcoxon signé
2 groupes appariés — données non normales
Code R
wilcox.test(avant, apres, paired = TRUE) # Avec p-value exacte (recommandé pour n < 25) : wilcox.test(avant, apres, paired = TRUE, exact = TRUE)
Taille d'effet
library(effectsize) rank_biserial(avant, apres, paired = TRUE) # r < 0.1 = trivial | 0.3 = moyen | 0.5 = grand
Test de Mann-Whitney U
2 groupes indépendants — données non normales
Code R
wilcox.test(shannon ~ habitat, data = df) # (wilcox.test = Mann-Whitney pour 2 groupes indép.) # Avec intervalles de confiance sur la différence : wilcox.test(shannon ~ habitat, data = df, conf.int = TRUE)
Taille d'effet — r de rang-bisériel
library(effectsize) rank_biserial(shannon ~ habitat, data = df)
Kruskal-Wallis
3+ groupes — données non normales, analogue non-param. de l'ANOVA
Code R
kruskal.test(shannon ~ habitat, data = df) # Résumé groupes (médiane ± IQR recommandés) : df %>% group_by(habitat) %>% summarise(med = median(shannon), iqr = IQR(shannon))
Si p < 0.05 → test de Dunn pour identifier les paires significatives
Post-hoc de Dunn
library(dunn.test) dunn.test(df$shannon, df$habitat, method = "bonferroni") # method = "holm" ou "BH" aussi acceptés
Taille d'effet — eta² de rang
library(effectsize) rank_eta_squared(shannon ~ habitat, data = df)
Corrélation de Spearman
Relation monotone — données non normales ou ordonnées
Code R
cor.test(df$shannon, df$altitude, method = "spearman") # Matrice de corrélations multiples : library(corrplot) M <- cor(df[, c("shannon","altitude","temp")], method = "spearman") corrplot(M, method = "circle", p.mat = cor_pmat(df[,vars]), sig.level = 0.05)
Spearman mesure la relation monotone (rang), pas forcément linéaire. Si la relation est clairement linéaire et les données normales → Pearson est plus puissant.
NMDS
Ordination non-métrique — visualiser les similarités de composition
Code R
library(vegan) # comm = matrice sites x espèces nmds <- metaMDS(comm, distance = "bray", k = 2, trymax = 100) # Stress : < 0.1 excellent | < 0.2 acceptable cat("Stress :", nmds$stress) # Figure avec ggplot2 library(ggplot2) coords <- as.data.frame(scores(nmds)) coords$site <- rownames(coords) ggplot(coords, aes(NMDS1, NMDS2)) + geom_point(aes(color = groupes), size = 3) + stat_ellipse(aes(color = groupes)) + theme_minimal()
NMDS ne teste pas les différences — c'est une visualisation. Combinez avec ANOSIM ou PERMANOVA pour tester.
ANOSIM + PERMANOVA
Tester si des groupes ont des compositions significativement différentes
Code R
library(vegan) # ANOSIM (Analysis of Similarities) anosim_res <- anosim(comm, grouping = habitat, distance = "bray", permutations = 999) # R proche de 1 = groupes très distincts # PERMANOVA (plus puissant, recommandé) adonis2(comm ~ habitat + temperature, data = env, method = "bray", permutations = 999) # Donne % de variance expliquée par chaque variable
Homogénéité de dispersion (tester d'abord)
disp <- betadisper( vegdist(comm, "bray"), habitat) permutest(disp) # Si p < 0.05 : dispersion inégale # → interpréter PERMANOVA avec précaution
PERMANOVA post-hoc : pairwise.adonis2() du package pairwiseAdonis
RDA / CCA
Relier la composition à des variables environnementales
Quelle méthode choisir ?
# Tester la longueur du gradient (DCA) dca <- decorana(comm) dca # Si axe 1 < 3 SD : RDA | > 4 SD : CCA
RDA (gradient court, réponse linéaire)
library(vegan) rda_res <- rda(comm ~ altitude + temperature + perturbation, data = env) summary(rda_res) anova(rda_res, permutations = 999) plot(rda_res) # biplot espèces + variables
CCA (gradient long, réponse unimodale)
cca_res <- cca(comm ~ altitude + temperature, data = env) anova(cca_res, by = "term", permutations = 999) # test par variable
SIMPER
Identifier les espèces qui expliquent les différences entre groupes
Code R
library(vegan) simper_res <- simper(comm, habitat, permutations = 999) summary(simper_res) # → liste des espèces ordonnées par contribution # à la dissimilarité entre chaque paire de groupes
Visualisation
# Top 10 espèces contributrices sp_contrib <- as.data.frame( summary(simper_res)$Foret_Savane) sp_contrib %>% slice_max(average, n = 10) %>% ggplot(aes(reorder(species, average), average, fill = average)) + geom_col() + coord_flip() + scale_fill_viridis_c() + theme_minimal()
SIMPER doit être accompagné d'un test ANOSIM ou PERMANOVA qui valide que les groupes diffèrent réellement avant d'interpréter.
GLM Poisson / Binomial négatif
Modéliser des comptages (abondance d'une espèce)
Code R — Poisson (pas de surdispersion)
model <- glm(abondance ~ habitat + altitude, family = poisson(link = "log"), data = df) summary(model) # Vérifier surdispersion deviance(model) / df.residual(model) # ratio > 1.5 → surdispersion → neg. binomiale
Si surdispersion → binomiale négative
library(MASS) model_nb <- glm.nb(abondance ~ habitat + altitude, data = df) summary(model_nb)
Les données de comptages avec beaucoup de zéros (>30%) nécessitent un modèle Zero-Inflated : pscl::zeroinfl()
Régression logistique
Modéliser la présence / absence d'une espèce
Code R
model <- glm(presence ~ altitude + temperature + couverture + perturbation, family = binomial(link = "logit"), data = df) summary(model) # Odds ratios (plus interprétables que les coefs) exp(coef(model)) exp(confint(model))
Performance du modèle
library(pROC) roc_curve <- roc(df$presence, fitted(model)) auc(roc_curve) # AUC > 0.7 = bon | > 0.9 = excellent
Pour les modèles d'occupancy (détection imparfaite) → utilisez le package unmarked avec occu()
