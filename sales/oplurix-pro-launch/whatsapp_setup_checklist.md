# EcoR Pro WhatsApp Setup Checklist

Updated: 2026-05-07

Use this to set up the private WhatsApp community for Pro buyers.

## Recommended Setup

- Group name:
  - `EcoR Pro - Buyers`
- Group description:
  - `Private support group for EcoR Toolkit Pro buyers. Focus: R workflow questions, biodiversity data handling, EcoR assets, and practical troubleshooting.`
- Visibility:
  - private invite only
- Admins:
  - start with just you
- Posting rules:
  - buyers may post questions
  - no unrelated promotion
  - no client-confidential data dumps
  - no expectation of real-time replies

## Founder Setup Steps

1. Create the WhatsApp group.
2. Add a short group description with the support boundary.
3. Pin one welcome message.
4. Save the invite link somewhere private.
5. Do not place the invite link on the public storefront.
6. Send the invite manually only after payment confirmation.
7. Record each invite in `pro_support_tracker.csv`.

## Pinned Welcome Message

```text
Welcome to EcoR Pro.

This group is for practical questions about the EcoR Toolkit Pro assets:
- scripts
- notebooks
- decision tree
- workflow use

Please do not post confidential client data, unpublished sensitive data, or anything that expects full custom analysis.

Email Q&A support and this group both follow the stated support boundary. I will reply as consistently as I can, but not in real time.

Haja Fabrice
```
