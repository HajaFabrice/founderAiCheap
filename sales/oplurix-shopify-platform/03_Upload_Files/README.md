# Upload Files

This folder contains the current OPLURIX files most relevant to a Shopify setup.

## Subfolders

- `Digital_Delivery_ZIPs/`
  - candidate digital delivery files
- `Product_PDFs/`
  - buyer-facing or preview-facing PDF assets already present in the repo

## Important Caution

These are not the same as Shopify product card images.

You still need:

- square or portrait product cover images in JPG or PNG format
- consistent thumbnail visuals for catalog browsing

Use this folder for:

- delivery attachments
- support/reference uploads
- internal store build staging
