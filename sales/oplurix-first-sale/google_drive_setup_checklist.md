# Google Drive Setup Checklist

Updated: 2026-05-03

Use Google Drive as the delivery host for the first sales cycle.

## Drive Structure To Create

1. Create a private folder:
   - `OPLURIX Deliveries`
2. Inside it, create:
   - `EcoR Complete`
3. Upload:
   - `EcoR_Toolkit_Complete_v1.zip`

## Sharing Rule

- Keep the main folder private.
- Use a shareable link only for the specific file you want the buyer to download.
- Prefer a link that lets the buyer view and download without exposing the rest of the Drive workspace.

## Delivery Rule

Before sending the link:
- confirm the PayPal payment
- confirm the correct buyer email
- confirm the final ZIP version is `EcoR_Toolkit_Complete_v1.zip`

In the delivery email:
- include the Google Drive link
- mention the `00_START_HERE_OPLURIX.md` file
- mention the support boundary note
- ask the buyer to confirm access

## Backup Rule

If the buyer cannot access the Drive link:
- resend with corrected sharing permissions
- or send the ZIP as an email attachment if size still allows
- or use a one-off backup transfer link manually
