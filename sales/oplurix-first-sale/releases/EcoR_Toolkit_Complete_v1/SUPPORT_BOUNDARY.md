# Support Boundary

This product is designed to help you move faster through biodiversity data
workflow setup.

It does not provide:
- formal statistical consulting
- project supervision
- custom analysis on your specific dataset
- automated interpretation you should trust without checking

Use the toolkit as a structured starting point, not as a replacement for
scientific judgment.

If the purchased package is missing a file or has a packaging issue, reply to
the delivery email and I will help directly.
