# Data Templates

Use these CSV templates as clean starting points for your own project.

- `transect_observations_template.csv` - main biodiversity observation table
- `site_metadata_template.csv` - site coordinates and habitat information
- `environmental_variables_template.csv` - environmental measurements by site/date
- `camera_trap_template.csv` - camera-trap detections
- `species_reference_template.csv` - canonical species names and taxonomic metadata

For best results, keep column names unchanged when using the R scripts.
