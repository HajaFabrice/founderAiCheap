# OPLURIX First Sale Workspace

Updated: 2026-05-03

This folder is the operational workspace for the first paid product download.
It is centered on `EcoR Toolkit Complete` at `USD 97`.

## First-Sale Defaults

- primary product: `EcoR Toolkit Complete`
- checkout: PayPal from the Netlify storefront
- delivery: manual email
- delivery window: within `12 hours` after payment confirmation
- delivery host: Google Drive
- launch motion: warm outreach first

## Folder Guide

- `release_support/`
  - buyer-facing files added to the delivery package
- `releases/`
  - generated release folder for `EcoR_Toolkit_Complete_v1`
- `google-drive-mirror/`
  - local mirror of the folder structure to upload to Google Drive
- `delivery_log.csv`
  - track payment, delivery, confirmation, and follow-up
- `warm_launch_list.csv`
  - separate warm product-launch list, not the NGO service CRM
- `ecor_complete_launch_messages.md`
  - human launch copy for the first buyer motion
- `google_drive_setup_checklist.md`
  - manual setup checklist for the Drive folder and sharing path

## First Buyer Workflow

1. Send warm launch messages to the `A` segment.
2. Route interested people to the storefront.
3. Verify the PayPal payment.
4. Send the Drive delivery email manually.
5. Log the delivery event.
6. Confirm the buyer downloaded the package.
7. Ask one short feedback question and record the answer.

## Important Boundary

- `EcoR Toolkit Pro Bundle` is not a live checkout product yet.
- Do not take payment for Pro until the walkthrough assets are actually ready.
- If a buyer asks for Pro, use the waitlist or request-access path instead.
