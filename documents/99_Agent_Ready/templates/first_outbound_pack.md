# First Outbound Pack

Updated: 2026-04-27

Use these only for clearly independent, non-Techni outreach. Every send still
requires human review and approval.

## English Set

Best fit:
- English-speaking biodiversity NGO, monitoring programme, or field-research team
- likely need: data cleanup, validation, or reporting support

Subject A:
- `Small data-cleanup support for your biodiversity monitoring workflow`

Subject B:
- `Free sample review for a monitoring dataset`

First-touch body:

```text
Dear [Name / Team],

I came across your biodiversity monitoring work and thought a small, practical offer might be useful.

I help conservation teams clean, validate, and structure biodiversity datasets so they are easier to use for analysis, mapping, and donor reporting. If useful, I can review a small sample of 5-20 rows and return three concrete fixes at no cost.

If the sample is helpful, I can then propose a short, bounded scope for cleanup or reporting support.

Would you like me to review a small sample?

Best regards,
Haja Fabrice Razafindrabe Maminiaina
```

Follow-up:

```text
Dear [Name / Team],

I am following up once in case the earlier note was useful but easy to miss.

If a small dataset sample would help, I can still review 5-20 rows and return a short note on the main cleanup or validation issues.

If this is not relevant for your team right now, no problem at all.

Best regards,
Haja Fabrice
```

Positive reply response:

```text
Hi [Name],

Thank you. Please send a small sample export and, if possible:
- the context of the dataset
- the variables that matter most
- the output you need most urgently

I will review the sample and return a short note covering the likely fixes, the questions that still need clarification, and the best next step for a bounded scope.

Best regards,
Haja Fabrice
```

## French Set

Best fit:
- Francophone NGO or biodiversity monitoring team
- especially where field data or reporting tables need practical cleanup

Objet A:
- `Petit appui pour le nettoyage de vos donnees de biodiversite`

Objet B:
- `Revue gratuite d'un petit extrait de donnees`

Premier message:

```text
Bonjour [Nom / equipe],

Je suis tombe sur vos activites de suivi biodiversite et je me suis dit qu'une proposition tres simple pourrait peut-etre vous etre utile.

J'aide des equipes de conservation a nettoyer, valider et structurer leurs jeux de donnees de biodiversite pour les rendre plus faciles a utiliser pour l'analyse, la cartographie et le reporting. Si cela vous aide, je peux examiner gratuitement un petit echantillon de 5 a 20 lignes et vous renvoyer trois corrections concretes.

Si ce retour est utile, je pourrai ensuite proposer un petit scope borne pour un nettoyage ou un appui de reporting.

Souhaitez-vous que je regarde un petit extrait ?

Bien cordialement,
Haja Fabrice Razafindrabe Maminiaina
```

Relance:

```text
Bonjour [Nom / equipe],

Je me permets une seule relance au cas ou mon precedent message aurait pu etre utile.

Si vous le souhaitez, je peux toujours examiner un petit extrait de 5 a 20 lignes et vous renvoyer une note courte sur les principaux points de nettoyage ou de validation.

Si ce n'est pas prioritaire pour vous en ce moment, aucun souci.

Bien cordialement,
Haja Fabrice
```

Reponse si interet:

```text
Bonjour [Nom],

Merci. Vous pouvez m'envoyer un petit extrait ainsi que, si possible :
- le contexte du jeu de donnees
- les variables les plus importantes
- le besoin le plus urgent cote analyse ou reporting

Je regarderai l'echantillon et je vous renverrai une courte note avec les corrections probables, les points a clarifier et la meilleure suite possible dans un scope borne.

Bien cordialement,
Haja Fabrice
```

## Guardrails

- No pricing in first-touch outreach.
- No employer-context leads.
- No client-name claims that are not grounded.
- If ownership or facts are unclear, stop and flag `NEEDS_HUMAN_VERIFICATION`.
