# EcoR Pro And Institutional Templates

Updated: 2026-05-07

## Pro Clarifier

```text
Pro is the higher-support individual EcoR tier. It is a real paid offer, but fulfillment is still founder-managed so the buyer receives the package, support-window details, and WhatsApp access steps honestly.
```

## Institutional Clarifier

```text
Institutional is a real team path, not a self-serve checkout. The next step is intake and a manual quote so the onboarding and adaptation promise stays truthful.
```

## Team Vs Individual Routing

```text
If you are asking as one researcher, Pro is the likely fit. If you are asking for a lab, NGO team, or department, the right path is Institutional.
```
